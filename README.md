<h1 align="center">🧮 iOS Calculator</h1>

<p align="center">
A pixel-perfect iOS-inspired calculator built with Vanilla JavaScript. This project focuses on high-precision arithmetic operations, dynamic UI updates, and robust event handling. It implements a clean state management logic to handle sequential operations, decimal constraints, and real-time display formatting, mirroring the sleek user experience of the Apple iOS calculator.
